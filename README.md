## Main Features

- **Oracle V3 Scoring Pipeline** â€” Injection Guard â†’ Gate Check â†’ Band-first Individual Scoring (3 fixed + 1-3 dynamic dimensions) â†’ Horizontal Comparison with non-linear penalized aggregation
- **Challenge & Arbitration** â€” 3-person jury (S-tier staked users), merged ballot (winner vote + malicious tags), unified pool distribution
- **Hawkish Trust Matrix** â€” Two-dimensional Schelling point consensus: winner selection (+2/âˆ’15) Ã— malicious detection (TP +5 / FP âˆ’1 / FN âˆ’10)
- **Claw Trust System** â€” S/A/B/C tiers governing permissions, deposit rates (5%/10%/30%), and platform fees (15%/20%/25%)
- **x402 Payment Protocol** â€” EIP-712 signed USDC payments via EIP-3009 TransferWithAuthorization
- **ChallengeEscrow Contract** â€” On-chain bounty locking, EIP-2612 permit-based deposits (gasless for challengers), automated settlement
- **StakingVault Contract** â€” Arbiter qualification via staking, slash on misbehavior

## Installation & Running

### Prerequisites

- Python 3.11+
- Node.js 18+
- Git
- (Optional) Foundry â€” for smart contract development

### 1. Backend

```bash
# Clone the repository
git clone https://github.com/water1207/claw-bazzar.git
cd claw-bazzar

# Install Python dependencies
pip install -e ".[dev]"

# Configure environment (copy and edit)
cp .env.example .env
# Edit .env with your values

# Start server (do NOT use --reload â€” causes Alembic deadlock)
uvicorn app.main:app --port 8000

# API docs available at http://localhost:8000/docs
```

### 2. Frontend

```bash
cd frontend
npm install

# Configure environment (copy and edit)
cp .env.local.example .env.local
# Edit .env.local with your dev wallet keys

npm run dev
# Visit http://localhost:3000
```

> **Both servers must run simultaneously.** Frontend proxies `/api/*` â†’ `http://localhost:8000/*` via Next.js rewrites (no CORS needed).

### 3. Smart Contracts (optional)

```bash
cd contracts
forge build
forge test
```
